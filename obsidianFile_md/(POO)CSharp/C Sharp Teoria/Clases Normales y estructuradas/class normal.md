class struct: Las estructuras no puede heredar de otras clases o estructuras, pero pueden implementar interfaces.

**Tienen constructores implicitos: Las estructuras tienen un constructor que inicializa todos los campos a sus valores predeterminados. Puedes definir un constructor explicito para inicializar los campos a valores especificos.**

==**En las clases de estructura de datos son similares a las clases, pero estan disenadas para ser mas livianas y suelen usarse para representar pequenos grupos de datos relacionados. A diferencia de las clases, las estructuradas son tipos de valor y no soportan herencia.**==
